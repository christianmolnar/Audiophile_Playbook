Audiophile Playbook by Christian Molnar

My AI-fueled, Gateway to Headphone Happiness
Ever wondered how to squeeze every last drop of joy from your DAPs and headphones? So did I!
I put this site together for two reasons:

1. To see how it feels to create a site in GitHub copilot by talking to an AI and writing no code. This site is the result. Took about 8-10 hours over 2 days. I did write some of the code because I wanted to, but I really wouldn't have had to.
2. To see how good AI is at solving a problem end to end.
THE CHALLENGE
You have a couple of Digital Audio Players and a number of different headphones. How do you get the most beautifully equalized music, to your taste, from any combination?
THE APPROACH
  1. Ask AI what it knows about the problem and how it would go about solving it.
  2. Evaluate the options it proposes and pick a good one.
  3. Ask AI to solve it end to end.
  4. Guide it, answer its questions, and see how far you get.
THE PROCESS
The assistant quickly grasped the problem and, unprompted, explained to me the complexities of attempting to re-create a sonic signature across DAPs that have different features and headphones that feature completely different sonic profiles.

Its proposal was smart: Let me analyze your DAPs and Headphones, and let's settle on a sonic profile, and I can come up with EQ 'tunings' to reproduce that same profile with any combination.

It also asked me questions about the type of music I listened to, and how I liked to listen to it, and this was its brilliant proposal: let's create 3 sonic profiles:
  1. Musical Warmth: warm and perfect.
  2. Max Bass Fun: musical, warm, but with enhanced bass and body.
  3. Ultimate Relax: smooth, lush and non-fatiguing for extended listening.
THE EXPERIENCE
It all started so easy! The nicer I was to it, the nicer it was back! Compliments on my project management skills, my vision and intelligence! :-)

Once we settled on the data, I asked it to give me an EQ setting for one of my headphones on the Kann Max and, in a neat table, I saw gains from 30Hz to 18k, in a very intuitive Harman curve. The AI explained to me where the deltas were, for example: this particular headphone has a spike at 6Hz, so I'm compensating for that. It was brilliant!

Next step was to talk about how to lay out the site and represent this. What you see in this site is truly a collaboration between me and the AI. A lot of what you see was proposed by the AI, and all of it was coded by GPT, once I moved to Visual Studio Code with the basic HTML that I got from the web browser.
THE TROUBLES
The problems started when I asked it to implement the code in the browser. It proposed that it could give me a zip file with THE WHOLE SITE! I got a nice, neat table with all steps, times for each, and total ETA: 30-45 minutes! How can I say no?!?!

Fast-forward 4 hours, 2 am, and still not one single line of code. It would go something like this:
  1. 1. I'm generating! Silence..... How's it going? ... Perfect timing! I'm just about ready! Silence... Hello? ... Ah, there you are! I have it for you! Here's the download link... Link expired, file not found, error after error.
  2. About the same as #1... File works.... File has a couple of lines, and a couple of stubs... I ask, 'Hey, this file is empty!'.... Silence ..... Response: You are absolutely correct. And it is my fault. My environment got reset so I didn't finish it. It was literally like me in highschool... Then it would apologize profusely, give me a plan, and go back to work, only to rinse and repeat.


THE MAGIC
  • Talking to a super intelligent and resourceful assistant that actually proposes smart solutions and builds upon your ideas is just so cool.
  • The REAL MAGIC happened in VS Code. I just couldn't put it down. Just talk to it and the code happens. Sentences like this one, which in any other context would just get back a big 'I have no idea what you are talking about' would give back perfect code:
    • Me: 'Another box, THE TROUBLES, with this text and the steps in a numbered list'
    • GPT would produce exactly what I asked for. It saw what I was doing with shades, so it would start making each box progressively darker. It saw I rounded corners, so it tells me 'Let me make the changes to make all major styling global. It will make it much easier for you to maintain' and then it would do it, and beautifully comment it...
  • And the list goes on.. That's just a simple example. Take a look at the graphs and tables. If you subtract my learning and all the fighting with the web GPT just to test the limits, I think this whole site probably took 3 hours.
  • The last page "Story" is a treat. Check it out. Just for fun. The animated gifs, EVERYTHING was created by me just telling it what I wanted...
